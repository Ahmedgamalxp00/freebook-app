
# FreeBook App

it's a very simple app showing you free books and allow you to read them , also showing related books and allow you to search for books .



## Features

- get books and read them
- get related books
- search for a book


## Technologies
- flutter / dart
- animation
- api
- http / dio
- version controll
- postman
